# Junior Security Analyst Intro

## Junior Security Analyst Journey
+ Peran Junior Security Analyst (SOC L1) adalah sebagai garis pertahanan pertama yang bertugas melindungi perusahaan dari berbagai ancaman siber setiap hari.

+ Gambaran utamanya:
  + Dunia cyber security penuh dengan ancaman yang terus berkembang (serangan, kebocoran data, vulnerability baru)
  + SOC L1 menjadi pihak pertama yang mendeteksi dan menganalisis ancaman tersebut

+ Tugas harian:
  + Monitor & investigate security alerts
  + Ikut diskusi/brainstorm SOC
  + Bekerja sama dengan tim lain (IT, security, dll)
  + Terus belajar tentang serangan dan pertahanan terbaru

+ SOC L1 adalah garda terdepan yang bertugas mendeteksi, menganalisis, dan membantu mencegah serangan agar perusahaan tetap aman
